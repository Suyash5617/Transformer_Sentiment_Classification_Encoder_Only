# Transformer Sentiment Classification (Encoder-Only)

A Transformer Encoder-based Sentiment Analysis project built using TensorFlow and Keras. The model uses Token Embeddings, Positional Encoding, and Multi-Head Self-Attention to classify text as Positive or Negative sentiment.

## Features
- Text preprocessing and tokenization
- Token Embedding and Positional Encoding
- Multi-Head Self-Attention mechanism
- Custom Transformer Encoder implementation
- Binary Sentiment Classification
- Model Training and Evaluation
- Sentiment Prediction on New Text Inputs

## Tech Stack
Python, TensorFlow, Keras, NumPy, NLP, Transformer Encoder, Multi-Head Attention

## Workflow
1. Dataset Creation
2. Text Vectorization and Tokenization
3. Token & Position Embeddings
4. Multi-Head Self-Attention
5. Transformer Encoder
6. Model Training
7. Model Evaluation
8. Sentiment Prediction

## Model Architecture

Input Text
→ TextVectorization
→ Token Embedding
→ Positional Encoding
→ Multi-Head Self-Attention
→ Transformer Encoder
→ Global Average Pooling
→ Dense Layer
→ Sigmoid Output
→ Positive / Negative Sentiment

## Sample Predictions

| Input Sentence | Prediction |
|---------------|------------|
| This session is good | Positive |
| Teaching is excellent | Positive |
| I did not like this course | Negative |
| This course is confusing | Negative |
